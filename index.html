    Material Deck SWADE Documentation

    This documentation is for the Savage Worlds Adventure Edition system module for Material Deck.
    Please refer to the Material Deck documentation for more info on Material Deck.

    The Material Deck SWADE system module adds SWADE-dependent features to Material Deck to the following actions:

    • Combat Tracker
    • Other Actions
    • Token

    Combatibility

    Foundry VTT: v12-13
    Savage Worlds Adventure Edition: v4-5
